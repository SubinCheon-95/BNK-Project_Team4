# BNK부산은행 외환상품 관리 시스템 1/2차 Project
## 📌 프로젝트 개요

| 구분 | 내용 |
|------|------|
| **프로젝트 구분** | 금융·은행권 웹&앱 서비스 개발 |
| **프로젝트 명** | BNK부산은행 외환상품 통합 관리 시스템 |
| **배경 및 목적** | - BNK부산은행의 외환 및 금융상품 정보를 통합 관리할 수 있는 웹·앱 서비스 구현<br>
                     - 실제 은행 서비스 구조를 반영하여 금융 데이터 설계, 관리, 보안 흐름을 경험<br>
                     - 디자인부터 백엔드까지 전 과정을 이해하고 협업 역량을 강화 |
| **기대 효과** | - 은행권 웹 시스템 개발 프로세스 실무 감각 향상<br>
                  - UI/UX 설계와 관리자 페이지 구축 경험을 통한 실전 대응 능력 강화<br>
                  - 데이터 중심의 시스템 관리 및 통계 기능 구현 경험 축적 |
| **주요 기능** | - 사용자 화면: 가입하기&로그인&로그아웃 / 외환상품 조회 / 상품상세 / 금융감정분석 / 번역(다국어)<br>
                  - 관리자 화면: 회원관리 / 상품관리 / 게시판관리 / 고객센터관리 / 환경설정<br>
                  - 시각화 기능: 판매통계, 조회통계, 상품유형 비율 원형그래프, 감정분석 리포트 |
| **서비스 채널** | PC WEB / MOBILE WEB (반응형) |
| **개발 환경** | - Frontend: HTML, CSS, JavaScript, Adobe XD<br>
                  - Backend: Java 17, Spring Framework(Spring Boot 3.x), MyBatis<br>
                  - Database: Oracle 21c XE<br>
                  - Server: AWS EC2 / Tomcat 10<br>
                  - 협업/CI: GitHub / Figma / Notion |
| **작업 기간** | - 1차: 2025.11.06 (목) ~ 2025.12.03 (수) / 160h<br>
                  - 2차: 2025.12.05 (금) ~ 2026.01.06 (화) / 160h |
| **작품 발표일** | - 1차: 2025.12.04(목) 09:00<br>
                    - 2차: 2026.01.06(화) 14:00 |
